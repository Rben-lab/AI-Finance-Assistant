# AI-Powered Personal Finance Assistant & Budget Planner

This project is an AI-based financial dashboard developed using Streamlit and Google Gemini API.

## Features

* Budget Allocation Planner
* Expense Analysis
* Financial Health Score
* CSV Transaction Analysis
* AI Financial Advisor
* AI Financial Chatbot

## Tech Stack

* Python
* Streamlit
* Pandas
* Google Gemini API
* Matplotlib

## Run Project

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Streamlit:

```bash
streamlit run app.py
```
